Thanks for installing Summarize AI 🎉

This extension helps you quickly create concise summaries of web pages, selected text, and more using powerful AI models.

  • Highlight any text and click the toolbar button or right‑click & choose "Summarize selection".
  • Open the popup (Ctrl+Shift+U) to manage summaries, bookmarks, and settings.
  • Configure your API key and preferences on the Settings page.
  • Use keyboard shortcut Ctrl+Shift+S (or ⌘ on mac) to summarize selected text instantly.
